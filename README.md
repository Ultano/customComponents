# Custom Components Collection

This repository contains a set of lightweight web components implemented using the native Custom Elements API. They provide a starting point for building dashboards or control panels with light and dark themes.

## Components
- `custom-icon` – displays simple icons.
- `custom-card` – card container with header and body.
- `custom-menu` – horizontal navigation menu.
- `custom-panel` – panel/section container.
- `theme-toggle` – button to switch themes.

See individual component folders under `components/` for detailed documentation.

## Example Application
Open `examples/dashboard/index.html` in a browser to see the components in action.
