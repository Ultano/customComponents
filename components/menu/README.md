# custom-menu

Horizontal navigation menu using the primary theme color.

## Usage
```html
<script type="module" src="../../components/menu/custom-menu.js"></script>
<custom-menu>
  <a href="#home">Home</a>
  <a href="#about">About</a>
</custom-menu>
```

## Slots
- default: anchor links or buttons

Style the links inside as desired.
